# 卡牌記憶遊戲

## 使用說明:
在遊戲開始前，會先跳出一個選擇難度的視窗
 
<img width="833" height="750" alt="image" src="https://github.com/user-attachments/assets/2f9b66da-33cd-4ac5-8265-890518402db3" />

選完難度後，視窗會跳到對應的關卡，並播放遊戲開始的語音提醒玩家，下圖為中等難度的畫面
 
<img width="865" height="675" alt="image" src="https://github.com/user-attachments/assets/541eebe2-5d3f-422a-8a5e-ed51426a8dd0" />

點選牌背即可把牌翻開

<img width="865" height="671" alt="image" src="https://github.com/user-attachments/assets/b3c63365-44a6-433d-8c6e-e78933723111" />

選完兩張後，翻牌次數會加一，並檢查兩張牌是否相同，若是不一樣，牌會自動翻回變牌背;若是一樣，則翻出來的牌會用綠色框標示出來，並更新分數和進度

<img width="865" height="672" alt="image" src="https://github.com/user-attachments/assets/96ce5a89-6917-4c9b-ba00-949e91e2a6bb" />

若是連續兩次翻出來的牌都是相同，得到的分數會更多
全部的牌都配對成功，或是時間到遊戲就會停止。
